## 🖥️ Running Locally

Getting started is as simple as it gets — no installs, no setup, no terminal magic required.

**1. Clone the repository:**

```bash
git clone https://github.com/shawnazd/css-tutorial.git
```

**2. Navigate into the folder:**

```bash
cd css-tutorial
```

**3. Open in your browser:**

Just double-click `index.html` or drag it into any browser window. That's it — you're in.
